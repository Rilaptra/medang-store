## 📝 Sekilas Tentang Medang Market 🛍️

Bayangin deh, **Medang Market** itu kayak pasar digital 2.0 buat sekolah kamu! 🚀  Di sini, semua warga sekolah - siswa, guru, staf - bisa dengan mudah:

## 🌟 Fitur-Fitur Unggulan ✨

Medang Market punya segudang fitur kece yang bikin pengalaman jual beli di sekolah jadi makin asyik:

### 👤 Manajemen Pengguna yang Lengkap

* **Pendaftaran & Login Mudah:**  Proses daftar dan login super simpel dan cepat! ✅
* **Profil Pengguna Personal:**  Bikin profil kamu sendiri! 🖼️  Pamerkan nama, bio singkat, kelas, foto profil, dan info menarik lainnya!
* **Sistem Role Pengguna:** Ada tingkatan peran yang berbeda:
    * **Member:** Pengguna biasa untuk belanja. 🧑‍🎓
    * **Penjual:**  Bisa jualan produknya di platform. 🧑‍💼
    * **Admin:**  Pegang kendali penuh untuk manajemen platform. 👮
    Masing-masing role punya "kekuatan" dan akses yang beda! 💪
* **Admin Panel Canggih:**  Panel khusus admin untuk mengatur pengguna dan produk dengan mudah. 👮‍♀️

### 📦 Manajemen Produk yang Fleksibel

* **Penjual Bisa Tambah & Atur Produk:**  Penjual punya kebebasan penuh untuk menambahkan dan mengelola produk jualan mereka! ✍️
* **Variasi Produk:** Dukungan untuk variasi produk! Misalnya, jual kaos dengan pilihan ukuran dan warna yang beragam. 🌈
* **Detail Produk Lengkap:**  Unggah foto produk yang menarik 🖼️, atur harga yang kompetitif 💰, dan pantau stok barang secara _real-time_! 📊
* **Pencarian & Filter Kategori:**  Pengguna bisa dengan mudah mencari produk yang diinginkan dan memfilter berdasarkan kategori. 🔍  Gampang banget nemuin barang yang dicari!

### 🛒 Keranjang Belanja & Proses Checkout yang Mulus

* **Tambah ke Keranjang dengan Sekali Klik:**  Proses menambahkan barang ke keranjang belanja sangat intuitif. ➕
* **Manajemen Keranjang yang Mudah:**  Atur isi keranjang belanja dengan gampang! Ubah jumlah barang, hapus item yang tidak jadi dibeli. 🗑️
* **Checkout Otomatis & Cepat:**  Proses checkout yang simpel, total harga dihitung otomatis. 🧾  Bayar, selesai!

### 😎 Profil Pengguna yang Interaktif

* **Lihat Profil Pengguna Lain:**  Intip profil pengguna lain, lihat produk jualan mereka (jika mereka penjual), dan daftar _follower_ mereka. 👀
* **Fitur Follow/Unfollow:**  Bangun koneksi! Follow pengguna lain yang menarik minatmu. 🫂

### 🛡️ Panel Admin yang Powerful

* **Manajemen Pengguna Penuh:**  Admin bisa mengatur semua pengguna dan peran mereka. 🧑‍💼
* **Manajemen Produk Sentral:**  Admin punya kendali penuh atas semua produk yang ada di platform. 📦
* **Verifikasi Penjual:**  Proses verifikasi untuk pengguna yang ingin menjadi penjual. ✅  Menjaga kualitas dan keamanan platform.

### 🔎 Fitur Pencarian yang Cepat

* **Pencarian Berbasis Nama Produk:**  Cari produk dengan mengetikkan nama produk di kolom pencarian. ⌨️
* **Filter Berdasarkan Kategori:**  Saring hasil pencarian berdasarkan kategori produk. 🗂️  Lebih spesifik, lebih cepat!

### ✨ UI/UX yang Dinamis & Menawan

* **Desain Responsif:**  Tampilan website tetap keren dan berfungsi optimal di semua perangkat - desktop, tablet, maupun _smartphone_. 📱💻
* **_Dark Mode_ & _Light Mode_:**  Pilih tema tampilan sesuai preferensi!  _Light mode_ cerah atau _dark mode_ yang nyaman di mata. 🎨

### ⚙️ API Routes untuk Developer (Optional) 🚀

* **API Lengkap:**  Tersedia API untuk manajemen pengguna, produk, keranjang, dan pesanan dengan operasi CRUD (Create, Read, Update, Delete).  Khususnya buat kamu para developer yang ingin mengembangkan lebih lanjut! 🚀

## 🛠️ Teknologi yang Dipakai 💻🚀

Medang Market dibangun dengan teknologi web modern dan _powerful_:

**Frontend:**

* **[Next.js](https://nextjs.org/)**: Framework React untuk performa tinggi, SEO optimal, dan pengembangan yang menyenangkan. ⚛️
* **[React](https://reactjs.org/)**: Library JavaScript terpopuler untuk membangun antarmuka pengguna yang interaktif. 💻
* **[TypeScript](https://www.typescriptlang.org/)**:  JavaScript dengan _type checking_! Kode lebih aman, lebih mudah di-maintain. ⌨️
* **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS _utility-first_ untuk styling yang cepat dan konsisten. 🎨
* **[Lucide React](https://lucide.dev/)**: Library ikon-ikon SVG yang cantik dan ringan. ✨
* **[Zustand](https://zustand-demo.pmnd.rs/)**:  State management yang _simple_ dan _powerful_ untuk React. 🎛️
* **[React Hook Form](https://react-hook-form.com/)**: Library untuk manajemen form yang mudah dan efisien. 📝
* **[Next Auth](https://next-auth.js.org/)**:  Autentikasi yang aman dan mudah diimplementasikan untuk aplikasi Next.js. 🔑
* **[Sonner](https://sonner.emilkowalski.com/)**: Library untuk notifikasi _toast_ yang elegan. 🍞
* **[Next Theme](https://www.npmjs.com/package/next-theme)**:  Manajemen tema (_light/dark mode_) yang mudah. 🎨

**Backend:**

* **[Node.js](https://nodejs.org/en)**:  Runtime JavaScript untuk backend yang cepat dan scalable. 🚀
* **[MongoDB](https://www.mongodb.com/)**: Database NoSQL yang fleksibel dan cocok untuk aplikasi web modern. 🗄️
* **[Mongoose](https://mongoosejs.com/)**:  ODM (Object Data Modeling) untuk MongoDB di Node.js. Memudahkan interaksi dengan database. 🧰
* **[Bcryptjs](https://www.npmjs.com/package/bcryptjs)**:  Library untuk _hashing password_ yang aman. 🔒  Keamanan data pengguna adalah prioritas!

**Deployment:**

* **[Vercel](https://vercel.com/)**:  Platform _deployment_ favorit untuk aplikasi Next.js! ☁️ (atau platform _deployment_ sejenis)

## 🚀 Siap Mencoba? Ikuti Langkah Mudah Ini! 👇

1. **Clone Repository:**

   ```bash
   git clone https://github.com/usernamekamu/medang-market.git
   cd medang-market
   ```

2. **Install Dependensi:**

   ```bash
   npm install
   ```

3. **Konfigurasi _Environment Variables_:**

   * Buat file `.env.local` di direktori utama proyek. 📂
   * Tambahkan konfigurasi koneksi MongoDB (`MONGODB_URI`), _session secret_ (`NEXTAUTH_SECRET`), dan Cloudinary API keys (jika menggunakan fitur unggah gambar). Contoh:

     ```env
     MONGODB_URI=mongodb+srv://usernamekamu:passwordkamu@clusterkamu.mongodb.net/namadatabasemu
     NEXTAUTH_SECRET=secretkamu
     NEXTAUTH_URL=http://localhost:3000
     NEXTAUTH_URL_INTERNAL=http://localhost:3000
     CLOUDINARY_CLOUD_NAME=cloud_name_kamu
     CLOUDINARY_API_KEY=api_key_kamu
     CLOUDINARY_API_SECRET=api_secret_kamu
     ```

4. **Jalankan Server Development:**

   ```bash
   npm run dev
   ```

   Buka [http://localhost:3000](http://localhost:3000) di browser kesayanganmu dan saksikan keajaiban Medang Market! ✨
